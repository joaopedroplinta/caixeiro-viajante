# Caixeiro Viajante

## Descrição
Solução para o Problema do Caixeiro Viajante (TSP - Traveling Salesman Problem), um dos problemas clássicos de otimização computacional.

## Objetivo
Encontrar a rota mais curta que visita todas as cidades exatamente uma vez e retorna ao ponto de partida.

## Funcionalidades
- Algoritmo de força bruta para pequenas instâncias
- Heurísticas (nearest neighbor, 2-opt)
- Comparação de desempenho entre abordagens
- Visualização de rotas

## Requisitos
- Python 3.8+
- Bibliotecas: numpy, matplotlib

## Instalação
```bash
pip install -r requirements.txt
```

## Uso
```python
python tsp.py instancia.txt
```
